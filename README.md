# Spiritual Warfare Scripture Plan

An improved static PWA for a 30-day Christ-centered Scripture study. It includes daily readings, prayers, reflections, notes, memory verses, a prayer journal, local progress tracking, export/import, offline caching, and basic SEO files.

## Deploy

Deploy the contents of this folder as the site root on Netlify.

## Local Preview

```sh
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
